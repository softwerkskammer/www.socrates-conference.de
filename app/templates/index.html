SoCraTes 2012

2nd International Software Craftsmanship and Testing Conference in Germany

August 2-4 2012, Rückersbach, Code Retreat on August 5

What?

This event is about the sustainable creation of useful software in a responsible way.

There will be two days of

  • highly collaborative interactions
  • lightning talks
  • self-organized Open Space

The event will be much like a retreat. We will be there for 48 hours to collaborate and share ideas.

When?

August 2-4 2012

More precisely:

The SoCraTes conference takes place from the evening of Thursday August 2nd 2012 (about 5pm) through the evening on Saturday August 4th 2012.

The conference is followed by a Code Retreat on Sunday August 5th 2012 (until about 6pm). This is a separate, optional event which is not included in the conference.

Where?

Out in the country, at the lovely Seminarzentrum Rückersbach (German only website), one hour from Frankfurt (Main) airport.

There you will get your room and the event will happen there. We will rent the whole facility, so we can concentrate on what we like to do most.

Venue Address:

Kolpingstraße 1, 63867 Johannesberg

Who should come?

Everyone who is concerned with coding, testing, code quality, software craftsmanship...

  • For you, coding is more than writing text in an editor.
  • For you, testing is more than ensuring quality.
  • For you, testing is more than executing given scenarios.

How?

The conference will be run as a non-profit, low-cost, community-organized event. Brought to you as a joint-effort of all Softwerkskammer groups, the community of all German Software Craftsmanship groups.

Facilitation will be in English, sessions in other languages are welcome.
